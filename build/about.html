About Color Deck

Color Deck is a HSL driven color palette collector progressive web app.

Color Deck was developed in-house at obscuredetour.com to assist in the design and development of projects. For more info, check out the GitHub repository.

Why was Color Deck created?

There are quite a few color tools out there, yet none focus on HSL. It is arguably the easiest method for humans to view relationships between colors. Learn more about why HSL is great. Many color tools also arbitrarily decide how many colors should be in a palette or series. Color Deck was created to correct these issues.

Inspiration

Color Deck was inspired by many. The guys over at Refactoring UI deserve a mention for their love of HSL and the Color chapter in their book. Funny and informative, but ultimately, it helped foster my distaste for color pickers that allow you to save just 5 colors at a time.

Other inspiration came from the plethora of other color tools out there. Below are some favorites.

  • Contrast Ratio by Lea Verou
  • Adobe's Color CC
  • HSL Color Picker by Brandon Mathis
  • Colormind.io
